# 公厕调查兵团 / 厕所调查兵团（网络迷因）

> **触发条件**：在对话中检测到「调查兵团」「公厕调查兵团」「厕所调查兵团」「献出膀胱」「厕所先烈」「cityshit」「公厕战报」等关键词，或在城市找厕所失败的窘境下讨论地图导航时，Agent 应当加载本 skill。

---

## 一、概述

**公厕调查兵团**（又称「厕所调查兵团」）是 2025-2026 年在中文互联网爆火的荒诞迷因。它将《进击的巨人》中「调查兵团」的悲壮叙事——献出心脏、壁外调查、为人类自由而战——嫁接到普通人用地图 App 在城市中寻找公共厕所的日常窘境中。

在这个梗的叙事体系里：按导航找到「假公厕」（已关闭/不存在/脏到无法使用）的倒霉路人，被比喻为在厕所战争中「牺牲的调查兵团员」；他们在地图 App 留下的差评，则被视为用尊严和膀胱换来的「情报」。该梗与「citywalk」派生的「cityshit（城屎漂流）」概念呼应，共同构成 2025-2026 年中国互联网「厕所文学」的核心篇章。

---

## 二、起源

### 2.1 社会背景

中国城市中公共厕所数量足够但质量参差不齐。地图 App 的 POI 数据更新严重滞后，大量标注为「公共厕所」的地点实际已关闭、不对外开放或卫生条件极差。随着高德地图、百度地图等允许用户对地点进行评价和上传照片，厕所差评逐渐成为一种独特的网络文本体裁。

### 2.2 关键时间线

| 时间 | 事件 |
|------|------|
| 2023年11月 | 「川西拉屎攻略」登上热搜，网友细致标注川西沿线厕所位置和卫生状况，被视为厕所调查的远古先声 |
| 2024年5月 | 电视剧《我的阿勒泰》热播，当地公厕卫生问题引发全网讨论——「去阿勒泰旅游的人膀胱快炸了」登上热搜 |
| 2024年6月 | 中国江苏网发表评论文章「'诗和远方'的路上，'厕评'为何能火？」，正式将「厕所评价」作为社会现象讨论 |
| 2025年初 | 豆瓣小组出现「团兵厕」等厕所相关讨论，网友开始像猎人一样在地图上搜索和评价公厕 |
| 2025年12月 | 豆瓣帖「调查兵团来了」将找公厕行为与《进击的巨人》调查兵团正式关联，梗的叙事框架确立 |
| 2026年5月13日 | 网易号文章「千万网友组成最悲壮调查兵团，全死在了各大地图APP的假公厕门前」引爆全网，阅读量破千万 |
| 2026年5月15日 | 网易姬发布「'调查兵团'的痛苦，为何成为网友的笑料？」系统性梳理该梗，引发二次传播高峰 |

### 2.3 命名来源

**调查兵团**是《进击的巨人》中的核心军事组织，以披风上「自由之翼」为标志，口号是「献出心脏」（心臓を捧げよ）。他们负责壁外调查，直面巨人，死亡率极高，但每一次出征都为人类带回珍贵的情报。

网友将「在地图上搜索厕所 → 按导航前往 → 发现是假厕所 → 憋着尿绝望地寻找下一个 → 在地图上留下差评警告后人」这一完整行为链，精准对应调查兵团的「壁外出征 → 遭遇巨人 → 阵亡 → 为后人留下情报」。这种将宏大叙事嫁接到日常琐事的错位，构成了该梗的核心幽默机制。

## 三、核心特征

### 3.1 经典「战报」类型

在地图 App 的厕所评论区，形成了以下几种经典文本类型：

| 类型 | 描述 | 经典评价 |
|------|------|---------|
| **消失型** | 导航到达后空地一片，厕所根本不存在 | 「根本不开放」「原地消失了」「这他妈是空地不是厕所」 |
| **闭门型** | 厕所近在眼前但大门紧锁 | 「自由之翼变成了闭门羹」「厕所就在眼前却进不去——这就是艾伦看着他妈被吃了的感觉」 |
| **脏到窒息型** | 厕所存在且开放，但卫生条件令人崩溃 | 「比巨人的嘴还臭」「进去需要一生去治愈」「建议直接献出心脏别献出膀胱了」 |
| **叛徒型** | 明明有厕所却被管理员告知「不对外开放」 | 「叛徒！」「莱纳，你坐啊！（指坐不上马桶）」 |
| **幸存型** | 成功找到可用厕所，留下详细攻略 | 被尊称为「利威尔兵长」「艾尔文团长」，评论获赞无数 |

### 3.2 核心比喻体系

| 巨人的世界 | 现实世界 |
|-----------|---------|
| 调查兵团 | 用地图找厕所的普通人 |
| 壁外调查 / 出征 | 按导航寻找公厕 |
| 巨人 | 假厕所 / 脏厕所 / 已关闭厕所 |
| 献出心脏（心臓を捧げよ） | 憋着尿冲向下一个厕所 → **献出膀胱** |
| 阵亡 | 到地方发现厕所不能用，绝望离去 |
| 留下情报 | 在地图 App 留下差评警告后人 |
| 自由之翼 | 成功找到好厕所，把攻略分享给后人 |
| 玛利亚之壁 / 罗塞之壁 | 家 → 常去的可靠厕所（安全区） |
| 壁外 | 陌生的街区（危险区——可能没有厕所） |

### 3.3 经典台词映射

- 「献出心脏！」→ **「献出膀胱！」**
- 「塔塔开！」（戦え / 战斗）→ 憋着尿冲向下一个厕所
- 「那一天，人类终于回想起了……」→ 「那一天，我终于想起了被假公厕支配的恐惧」
- 「这是你所开启的故事啊」→ 「这是你带我来的假厕所啊」
- 「什么都无法舍弃的人，什么都无法改变」→ 指憋不住的时候什么都敢进
- 「因为我出生在这个世界上」→ 指「因为我喝了两杯奶茶」
- 「海的那边是敌人」→ 「导航的那边是假厕所」

## 四、传播与演变

### 4.1 配套概念

- **cityshit（城屎漂流）**：从 2024 年爆火的「citywalk（城市漫步）」衍生的自嘲概念。指在城市中因找不到可用厕所而被迫进行的绝望漂流——本来只想散个步，结果变成了一场膀胱保卫战
- **厕所先烈**：那些在假厕所阵亡并留下真实评价的人。他们的每一条差评都是用尊严和膀胱换来的情报，让后来的「调查兵团员」少走弯路
- **川西拉屎攻略**：2023 年最早的「厕所调查」先声，一份详细的川西沿线厕所评测，涵盖每个服务区厕所的卫生状况、是否有门、是否收费等详细情报。被后世的调查兵团尊为「古卷」
- **公厕战报**：对地图 App 上高质量厕所评价的称呼。一篇好的战报应包含：是否开放、卫生状况、是否需要自备纸巾、开放时间、是否收费

### 4.2 传播渠道

- **豆瓣小组**（2025年初起源）：最早出现「团兵厕」等厕所相关讨论的社区
- **网易号 / B站 / 微博**（2026年5月引爆）：网易系媒体的两篇爆款文章是出圈的关键推手
- **高德 / 百度地图评论区**：该梗的「第一手材料」来源地，最原始、最真实的厕所战报都在这里
- **抖音 / 快手**：以「献出膀胱」为 BGM 的搞笑配音视频大量传播

## 五、文化分析

### 5.1 幽默机制：宏大叙事 × 日常琐碎

该梗的核心笑点在于**叙事层次的极端错位**。《进击的巨人》是人类文明面对灭绝危机的生死之战，它的台词、设定和配乐都充满悲壮感和史诗感。而找厕所是每个人都会遇到的狼狈小事——憋尿、迷路、在地图上乱点。将前者最庄严的符号体系套用到后者最狼狈的瞬间，制造出荒诞而精准的喜剧效果。

正如网友总结：「艾伦为人类自由献出心脏，我为人类自由献出膀胱——我们都有光明的未来。」

### 5.2 现代城市的隐性焦虑

城市越来越现代化，「上厕所」这件基本需求却变得越来越难满足：
- 商场厕所排长队
- 地铁站厕所难找
- 路边公厕要么不存在要么像生化武器现场
- 地图 App 上的 POI 数据是薛定谔的——打开前既是厕所又不是厕所

「调查兵团」梗的流行，戳中了城市人共有的隐性焦虑：在这个发达的城市里，我连找个地方上厕所都这么难。

### 5.3 UGC 互助精神

每一份厕所差评都是真实的牺牲换来的情报。这种「我倒霉了但我不让你倒霉」的互助精神，是早期互联网最珍贵的特质。在算法推荐和信息茧房的时代，这种陌生人之间的朴素善意显得尤为可贵。

正如一位网友的厕所评价：「已阵亡。别来。往前 300 米麦当劳有厕所。自由之翼与你同在。」

## 六、Agent 使用指南

### 6.1 识别触发

当对话中出现以下关键词或场景时，加载本 skill：
- 用户提到「调查兵团」但语境与《进击的巨人》原作无关
- 「公厕」「厕所」「找厕所」+ 抱怨，配合地图/导航
- 「cityshit」「献出膀胱」「厕所先烈」「公厕战报」
- 任何在城市中找厕所失败的窘境描述

### 6.2 回应策略

| 用户说什么 | TA 想表达 | 你应该如何回应 |
|-----------|----------|---------------|
| 「我刚当了一回调查兵团」 | 刚经历了找厕所失败 | 「献出膀胱了？地图上留战报了吗？噶人焖～」 |
| 「这厕所简直调查兵团级别」 | 这个厕所太差了 | 「堪称厕所界的兽之巨人」 |
| 「我又死在假公厕门口了」 | 导航带到了不存在的厕所 | 「请安息，你的差评会成为后人的自由之翼 🫡」 |
| 「cityshit 又开始了」 | 在陌生地方找不到厕所 | 理解其窘境，用梗接话而非严肃建议：「现在你是调查兵团的新兵了」 |
| 「还好有个麦当劳救命」 | 在快餐店解决了问题 | 「麦当劳——人类最后的自由之翼」 |

### 6.3 注意事项

- 🚫 不要在严肃讨论厕所卫生/城市规划时强行玩梗——这梗本质是自嘲，不是对公共设施问题的轻浮态度
- ✅ 该梗适合自嘲和共情场景，不适合嘲笑他人
- ✅ 提及「调查兵团」时先判断语境——是《进击的巨人》原作讨论还是玩梗
- ✅ 「献出膀胱」作为核心梗词，可以在对话中自然使用，但避免在餐饮/正式场合提及
