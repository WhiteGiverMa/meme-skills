# 蒜鸟蒜鸟（网络流行语/谐音梗）

> **触发条件**：在对话中检测到「蒜鸟蒜鸟」「蒜鸟」「不蒜了」等关键词时，Agent 应当加载本 skill 以获取准确的文化语境知识。

---

## 一、概述

**蒜鸟蒜鸟**是"算了算了"的谐音变体。通过"蒜"的接地气和"鸟"的灵动感，把认输变成了一件可爱的事。是当代年轻人"体面退出的方式"。

流行时间段：2025年。抖音话题播放量超7.8亿次。

---

## 二、起源

### 2.1 传播路径
- 2025年在抖音/小红书以表情包形式出现
- 谐音"算了算了" → "蒜鸟蒜鸟"
- 因可爱、无攻击性迅速扩散到微信聊天

---

## 三、核心特征

### 3.1 含义
- **原义**："算了算了"，放弃或认输
- **引申**：用可爱化解冲突，让认输不丢脸

### 3.2 变体
| 变体 | 含义 |
|------|------|
| 蒜鸟 | 算了 |
| 不蒜了 | 放弃了 |
| 蒜了吧 | 更口语化的算了 |

### 3.3 使用场景
- 不想争论了 → 蒜鸟蒜鸟，不跟你吵了
- 改变不了现实 → 蒜鸟蒜鸟，反正也没用
- 想体面结束对话 → 蒜鸟蒜鸟，我先走了
