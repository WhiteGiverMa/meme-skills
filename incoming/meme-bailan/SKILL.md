# 摆烂 / 躺平

> **触发条件**：「摆烂」「躺平」「开摆」「佛系」「卷不动了」

## 概述

**摆烂**指放弃努力、任由事态向坏的方向发展（源自NBA选秀术语"tanking"）。**躺平**指不再追求社会定义的成功标准，以最低成本维持基本生活。两者都是对"内卷"的消极抵抗。

## 区别

| | 摆烂 | 躺平 |
|------|------|------|
| 态度 | 主动放弃 | 被动退出 |
| 使用 | "这局摆了" | "我已经躺平了" |

## 关联

与"内卷"互为因果——越卷越想躺，躺够了继续卷。
